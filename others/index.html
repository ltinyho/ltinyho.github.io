项目经验

工作期间独立开发和主导了多个项目。

IM 客服系统

为了满足市场营销需求，负责重构微信公众号客服系统。前端使用 vue、 element开发， 后端 go 连接 websocket 进行通信。编写分表脚本，将原有 6000w 数据的大表分表，提高数据库查询、更新效率。发送消息的响应时间从 3s 提升到 300ms。

go 爬虫

为了第一时间获取考试成绩，使用 go 编写爬虫，通过定时任务和 mq 将请求并发数控制在一定范围内查询考试分数，第一时间获取到考试成绩并通知用户。

docker swarm 到 kubernetes 迁移

由于阿里云docker swarm容器服务不再提供服务，负责将阿里云原 docker swarm 到 kubernetes 容器服务的迁移。快速阅读 kubernetes 官方文档，规划整体迁移计划，负责集群的建立，编排模板的转化，容器应用的创建，测试，实现应用的平滑迁移。

优惠券模块

为了满足市场营销需求，独立设计、实现优惠券从建立、发放到使用、统计等功能。将原来需要手动编写代码进行改价的方式功能化。使用优惠券下单占比 15%，使用优惠券成单率 90%。

在线题库

为了适应业务的快速变化，主导在线题库模块的重写。前端使用 vue 框架开发，重构答题系统，实现一套代码，适用手机端和电脑端，增强了业务的灵活性，减少了 30% 的开发工作量和维护成本。同时引入 typescript 强类型、 jest 进行单元测试，增强代码的可维护性，健壮性，提高了系统的稳定性。
后端使用 go 开发，使用 redis 实现 mq 记录答题逻辑，将做题与统计解耦。

手机端学习平台

负责从零搭建手机端学习平台，将原来 jquery 项目整体迁移到 vue 单页面应用,为公司引入 vue 技术栈，提高了项目的开发效率和可维护性。主要实现商城购买、课程学习等功能。
后端使用 php、 go 对接阿里云视频点播、oss等系统，实现视频、图片文件的存储及播放。

接口生成器
